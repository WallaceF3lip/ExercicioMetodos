# 🧠 Exercícios Complementares — Métodos em Java

Este repositório contém uma atividade complementar com foco em **orientação a objetos**, criação de **classes**, **atributos** e **métodos**.

O objetivo é praticar conceitos fundamentais que serão utilizados nas próximas aulas.

---

## 📌 Objetivo da Atividade

Desenvolver um projeto simples para representar funcionários em uma aplicação, aplicando conceitos básicos de POO:

- Criação de classes
- Instanciação de objetos
- Encapsulamento
- Métodos com comportamento

---

## 🗂️ Nome do Projeto

```bash
ExercicioMetodos
```

---

## 📚 Exercícios

### 1️⃣ Classe `Funcionario`

Implemente uma classe chamada `Funcionario` contendo:

- Atributo para o **nome**
- Atributo para o **salário**

---

### 2️⃣ Classe `TestaFuncionario`

Crie uma classe chamada `TestaFuncionario` e:

- Instancie **dois objetos** da classe `Funcionario`
- Atribua valores aos atributos
- Exiba as informações na tela

---

### 3️⃣ Métodos na classe `Funcionario`

Adicione dois métodos:

- 📈 Um método para **aumentar o salário**
- 🔎 Um método para **consultar os dados do funcionário**

---

### 4️⃣ Testes dos Métodos

Na classe `TestaFuncionario`, teste os métodos criados utilizando um objeto da classe `Funcionario`.
